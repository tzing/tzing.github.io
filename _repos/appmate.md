appmate is a tool to quickly build a backend for prototype app.

We combined [rest framework] for the RESTful api, used [yasha] for quick setup
with [django] and its built-in ORM for sqlite. As the result, we could build
the backend by writing schema and deploy it to any device by just copy the
folder.

[rest framework]: www.django-rest-framework.org/
[yasha]: https://github.com/kblomqvist/yasha
[django]: https://www.djangoproject.com/
